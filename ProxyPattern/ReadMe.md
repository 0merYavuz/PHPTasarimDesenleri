# Proxy Pattern

Kodların Çıktısı:
``` 
Vatandaş: Başbakanla görüşmek istiyorum
Başbakanlık Kalemi: Tabii efendim, sizi başbakana yönlendiriyorum.
Vekil Başbakan: İş isteğinizi dinliyorum



Vatandaş: Başbakanla görüşmek istiyorum
Başbakanlık Kalemi: Tabii efendim, sizi başbakana yönlendiriyorum.
Vekil Başbakan: Derdinizi dinliyorum.
Başbakan: Dinliyorum

```
